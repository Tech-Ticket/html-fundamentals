# TechTicket – HTML Fundamentals

An interactive beginner-friendly course designed for students aged **11–17** to learn how websites are built using **HTML**.

This course introduces the fundamentals of web development through **visual explanations, interactive examples, and practical exercises**.
Each lesson is presented as an **interactive web page** so students can easily review the material after class.

---

## 🚀 Start the Course

Open the course here:

https://tech-ticket.github.io/html-fundamentals/

---

## 📚 Course Overview

The **HTML Fundamentals** course teaches students how web pages are created and how the web works.

Students will learn:

* How the Internet and the Web work
* What HTML is and how browsers read it
* HTML elements and attributes
* Creating links and images
* Organizing content using lists and tables
* Using HTML entities
* Structuring pages using layout elements
* Building a simple personal website

By the end of the course, students will build their **first real website using HTML**.

---

## 🧠 Course Sessions

The course contains **4 interactive sessions**.

1. **Session 1 — Understanding the Internet, the Web, and HTML**
2. **Session 2 — HTML Elements, Attributes, Links, and Images**
3. **Session 3 — Lists, Tables, and HTML Entities**
4. **Session 4 — HTML Layout, div, span, and Semantic HTML**

Each session includes:

* Concept explanations
* Visual diagrams
* Code examples
* Interactive exercises
* Practice challenges
* Homework tasks
